# magic-plugins   
magic-api 的右侧 资源插件扩展

## 基础信息

- **源码部分**: 包含了全部的插件开发过程所需的源码 
- **注意事项**: 请根据我写的样例，一点点替换自己的新的插件内容（如果出现资源 连接测试 按钮失效，请联系我，给你们发一个 前端包替换以下就ok了）
